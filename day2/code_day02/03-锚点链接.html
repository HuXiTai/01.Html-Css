早年经历

1992年，古力娜扎出生在新疆乌鲁木齐的一个维吾尔族传统家庭 [5] 。从小娜扎便被父母送到寄宿的私立艺术学校（新疆艺术学院）进行学习 [6] 。 虽天生是个美人坯子，但她却被父母给予了平凡的理想，在父母看来娜扎应该念书、上大学、找个好工作。而一次意外的机会让娜扎接触到了舞蹈，并开始一发不可收拾，随着年龄的增长，她也渐渐将舞蹈作为自己的梦想并为之努力 [7] 。

2007年，古力娜扎参加了新疆雪莲杯“和谐中华·中国第二届民族之花选拔活动”，而她也最终摘取了活动的维吾尔族之花 [8] ；之后，她还代表新疆参加了北京奥运会倒计时《和谐中华》晚会的民族服装展示。

2008年，一次偶然的机会让娜扎在街头被挖掘，兼职做起了平面模特。期间，她经常参加乌鲁木齐本地的一些服装发布会，并为一些影楼拍摄平面宣传照 [9] ，而这也渐渐让娜扎在行业内有了一些知名度 [7] 。

2009年，古力娜扎从新疆艺术学院中专部舞蹈系毕业；并于同年参加了中国职业模特大赛，而她在比赛中则最终获得了最上镜选手奖的荣誉；随后，她也获得了越来越多赴内地拍摄平面广告的机会 [9] 。

2010年，古力娜扎以素人身份参加了江苏卫视的《幸福晚点名》节目，她也在主持人李艾的建议下参加了陆川执导的电影《王的盛宴》“虞姬”一角的全国海选，并成为了候选者之一 [9] ，最终入围到了活动的16强 [10] 。此后，随着高考的临近，娜扎暂停了模特工作，为进入新疆军区文工团做准备，然而却因为地方文艺团体改制导致她错失了这个机会 [11] 。

演艺经历

2011年，在家待业的古力娜扎在朋友的鼓励下，参加了一年一度的艺术考试 [11-12] ，并因一张素颜照在网络上获得了颇高关注度 [9] ；随后，娜扎顺利考入了北京电影学院表演系。8月，她获得了与胡歌、刘诗诗、蒋劲夫等人合作出演古装玄幻偶像剧《轩辕剑之天之痕》的机会 [13] ，而该剧也是古力娜扎的影视处女作 [14] 。12月9日，娜扎签约上海唐人电影制作有限公司 [15] ，并由此正式踏入演艺圈。

2012年7月，古力娜扎出演的《轩辕剑之天之痕》开始在湖南卫视播出，其在剧中则饰演了美丽脱俗、思想单纯，且怀有宽容和怜悯之心的女娲之女小雪 [6] [16-17] ；随后，她还领衔主演了现代剧情片《隐私保险箱》，并在片中饰演了清纯可爱的女主角陈曼 [18] ，而这也是其个人的电影处女作；同年，娜扎还获得了第3届乐视影视盛典最佳新人奖的提名 [19-20] 。

2013年，古力娜扎先是与成龙、景甜等人共同出演了都市警匪片《警察故事2013》 [21-22] ；随后，她还与赵又廷、黄渤、林更新合作出演了现代动作片《痞子英雄之黎明升起》，其在片中饰演的则是崇尚正义且充满朝气的见习警花李小牧 [23] ；12月17日，娜扎还参加了第4届国剧盛典，并与其他几位嘉宾共同表演了盛典的开场秀 [24] 。

2014年，古力娜扎与刘烨、赵文卓、秋瓷炫等人合作主演了现代悬疑推理片《全城通缉》，而她在片中则饰演了叛逆独立的少女白晓若 [25] ；随后，其与邓超、杨幂等人共同出演的都市爱情喜剧片《分手大师》在全国上映 [26] ；同年，娜扎还与众多艺人联袂出演了现代爱情喜剧片《爱我就陪我看电影》 [27] ；此外，她还友情出演了青春校园励志剧《旋风十一人》 [28] 。

2015年，古力娜扎主演了古装玄幻爱情剧《青丘狐传说》，而其在剧中饰演的则是美艳聪慧的千年九尾狐陶恒；随后，她还参加了湖南卫视明星生活体验秀节目《偶像来了》 [29] ；同年，娜扎还与刘恺威、孙艺洲联袂主演了现代都市爱情剧《柠檬初上》，并在剧中饰演了乐观奔放、热情开朗，且坦率真诚的女主角宁小檬 [30-31] ，而该剧也是其个人的时装剧首秀。

个人生活

古力娜扎在家中排行老二。她的姐姐名叫古力加那提，比娜扎大5岁，职业是一名空姐。两人感情很好。2016年，姐姐在微博发文斥责了在网络上恶意攻击娜扎的网友 [69] 。

大学时期，古力娜扎曾与谢思宇交往，后分手 [70] 。2015年8月9日，古力娜扎与男演员张翰正式公布恋情 [36] 。2017年10月，古力娜扎与张翰和平分手 [71] 。

社会活动

2013年4月28日，古力娜扎参加了四川卫视“中国爱420芦山强烈地震公益特别节目” [86] 。 2016年2月，古力娜扎作为嘉宾出席了米兰时装周；9月1日，她还是参与了#画出生命线#公益直播活动，活动由参与者手绘救护车，呼吁更多社会人士筹集善款转化为真实救护车。在中国有592个国家级的贫困县，这些地区的卫生院求医就诊都非常困难，而娜扎此次直播所有收益都将转化为真实的救护车，用于改善偏远地区的救护支持 [87] 。

2017年9月，古力娜扎受邀再次参加了米兰时装周，并担任了Fendi等时尚品牌的头排观秀嘉宾 [88] ；同月，她还作为时尚品牌Tory Burch的唯一受邀艺人参加了春夏纽约时装周 [89] ；与此同时，古力娜扎还参与了芭莎慈善夜的捐款活动，虽然因故未能出席晚会，但她还是捐献了一个价值70万元人民币的救护车 [90] 。次年，她又再次参与了该活动，虽然仍未到场，但娜扎依然捐出了价值60万元人民币的六所希望小学 [91] 。

2019年3月26日，古力娜扎在上一年度的中国慈善名人榜中位列第20 [92] ；与此同时，她还参与了地球一小时活动，并担任了WWF明星志愿者 [93] ；11月16日，娜扎出席芭莎慈善夜 [94] ，并与众多艺人共同为扶贫项目“思源工程”进行了捐款 [95] ；同年，她还连续第二年入选北京师范大学新媒体传播研究中心、责任云研究院、封面新闻、封面新闻研究院、中国社会责任百人论坛联合发布的《中国影视明星社会责任研究报告》综合排行榜，排名分别为第92和第85位 [96-97] 。

2020年12月，古力娜扎担任了时尚芭莎慈善宣传大使 [98] ；随后，她还获得了中华思源工程扶贫基金会脱贫攻坚爱心个人的荣誉 [99-100] 。

2019年3月26日，古力娜扎在上一年度的中国慈善名人榜中位列第20 [92] ；与此同时，她还参与了地球一小时活动，并担任了WWF明星志愿者 [93] ；11月16日，娜扎出席芭莎慈善夜 [94] ，并与众多艺人共同为扶贫项目“思源工程”进行了捐款 [95] ；同年，她还连续第二年入选北京师范大学新媒体传播研究中心、责任云研究院、封面新闻、封面新闻研究院、中国社会责任百人论坛联合发布的《中国影视明星社会责任研究报告》综合排行榜，排名分别为第92和第85位 [96-97] 。
