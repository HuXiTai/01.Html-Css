一、CSS3新样式

1、圆角边框

在 CSS3 中，新增了圆角边框样式，这样我们的盒子就可以变圆角了。

border-radius 属性用于设置元素的外边框圆角。

语法：

 border-radius:length;
  • 参数值可以为数值（单位是px）或百分比的形式
  • 如果是正方形，想要设置为一个圆，把数值修改为高度或者宽度的一半即可，或者直接写为 50%
  • 该属性是一个简写属性，可以跟四个值，分别代表左上角、右上角、右下角、左下角
  • 分开写：border-top-left-radius、border-top-right-radius、border-bottom-right-radius 和border-bottom-left-radius
  • 兼容性 ie9+ 浏览器支持, 但是不会影响页面布局,可以放心使用

扩展：生成圆角的原理：生成了指定水平半径和垂直半径的圆形或椭圆形。

在定义一个角的角度的时候实际上定义的是水平和垂直的半径。

border-radius:50px 50px 50px 50px / 25px 25px 25px 25px;

border-raidus:左上水平x 右上角水平x 右下角水平x 左下角水平x / 左上垂直y 右上垂直y 右下垂直y 左下垂直y

4、背景相关

4.1 裁剪背景

默认的时候背景从内边距的外边界开始，平铺时一直平铺到边框的外边界。

格式：background-clip来控制背景延伸到哪里。

  • border-box，默认值，延伸到边框的外边界。
  • padding-box，延伸到内边距的外边界。
  • content-box，将背景限制在元素的内容区。
  • text，以区块中的文字作为裁剪的区域，此时应该用-webkit-background-clip:text这个属性。

案例：

4.2 背景定位基准

默认的时候background-position的定位相对于元素的内边距的外边界。使用background-origin来进行更改。

  • border-box，背景图像的左上角放在边框的左上角。
  • padding-box，放在内边距的外边界。
  • content-box，背景图像的左上角相对于内容区的左上角。

background-clip和background-origin区别：取值都相同，background-clip用来定义背景平铺时延伸到哪里。background-origin用来定义背景不平铺时从哪里开始，使用background-position时定位的基准是哪里。

4.3 调整背景图的尺寸

格式：background-size：value

必须设置两个值：第一个值是横向尺寸，第二个值是纵向尺寸。

  • px，像素值。
  • %，按照元素的宽度和高度来进行计算。
  • cover，等比例缩放到完全覆盖容器，背景图有可能超出容器。
  • contain，将背景图像等比例缩放到宽度、高度与容器的宽度或高度相等，背景图片始终被包含在容器内。

cover和contain的相同点和不同点：

  • 相同点：图片都是等比例缩放。
  • 不同点：cover是铺满整个显示区域，如果显示比例和显示区域比例相差比较大不会显示全（有可能会超出）。contain，按照某一边来覆盖显示区域，有可能会有没有覆盖到的地方（有可能会产生白边）。

5、滤镜

filter CSS属性将模糊或颜色偏移等图形效果应用于元素（不一定是图片），用来实现图形的各种各样的效果。

语法：

filter: 函数(); -->  例如： filter: blur(5px);  -->  blur模糊处理  数值越大越模糊

可选值一共有：

  • none，默认，没有效果。
  • blur()，模糊度。0为不模糊。使用px作为单位。值越大越模糊。
  • grayscale()，调整元素的灰度，值为0时没有变化，值为1时完全变成灰度图像。
  • opacity()，设置透明度，0表示完全透明，1表示不对元素做任何修改。
  • sepia()，调整元素的红褐色，值为100%是全褐色的，值为0时没有变化。
  • invert()，反向处理像素。0%时没有变化，100%时颜色反转。
  • brightness()，用来调整亮度，0%时为纯黑色，100%时元素没有变化。大于100%时颜色比元素原有的亮度更亮。
  • contrast()，用来设置元素的对比度。对比度越高越容易区分颜色。值为0%的时变成纯灰色，100%的时候没有变化，大于100%时才会得到更大的对比度。
  • filter: saturate()，调整元素上的饱和度，饱和度越高，颜色越鲜艳。调整为0%时没有饱和度，得到灰色。100%时，元素没有变化。

可以使用多个滤镜方法：filter: blur(2px) grayscale(100%);

6、opacity属性

​ 该属性指定了一个元素的不透明度，当该属性被应用于某个元素上时，这个元素及其所有子元素也会继承opacity属性。它的值也是从0~1的范围，0完全透明，1完全不透明， 默认值是完全不透明。 但当父元素设置opacity，子元素也设置opacity，在Firefox和Chrome下，子元素最终的opacity=父元素opacity*子元素opacity。

opacity使用注意点

opacity属性取值是从0~1,0完全透明，1完全不透明。

  • filter:opacity，一般用在图片上，因为filter可以有多个效果。
  • opacity属性，通常用在动画上，用来做整体的淡入淡出。
  • rgba背景透明，但是里面的内容不会透明。

二、常用函数

calc 函数

calc() 此CSS函数让你在声明CSS属性值时执行一些计算

语法：

width: calc(100% - 80px);

calc()使用通用的数学运算规则，但是也提供更智能的功能：

  1. 使用“+”、“-”、“*” 和 “/”四则运算；
  2. 可以使用百分比、px、em、rem等单位；
  3. 可以混合使用各种单位进行计算；
  4. 表达式中有“+”和“-”时，其前后必须要有空格，如"widht: calc(12%+5em)"这种没有空格的写法是错误的；
  5. 表达式中有“*”和“/”时，其前后可以没有空格，但建议留有空格；

attr函数

​ attr() 函数返回选中选择元素的属性值，支持版本：CSS2。
