2. 2d移动 translate

2d移动是2d转换里面的一种功能，可以改变元素在页面中的位置，类似 定位

使用2d移动的步骤：

  1. 给元素添加 转换属性 transform
  2. 属性值为 translate(x,y) 如 transform:translate(50px,50px);
  div{
    transform: translate(50px,50px);/*如果只有一个值，那么它会假定y的值为0。*/
  }

2D变形方法

  • translateX()，沿着元素自身的x轴移动。

    • px，像素。
    • %，相对于自身的宽度来计算（包括内容区，边框，内边距）

  • translateY()，沿着元素自身的y轴移动。

    • px，像素。
    • %，相对于自身的高度来计算（包括内容区，边框，内边距）

    注意：

    1. 多个变形方法一个接着一个，中间使用空格分隔。
    2. 多个变形方法一次只处理一个，从第一个开始一直到最后一个。
    3. 多个变形方法一定要设置正确，确保全部有效，其中一个出错，整个都失效了。

    translateX和translateY我们可以用来进行水平、垂直居中：

    当你使用margin和position进行定位时因为margin的百分比使用的是外层盒子的宽度作为标准，所以在使用margin时必须指定要居中的元素的一半的值。如果不知道要居中（水平、垂直）元素的宽度和高度这个我们可以使用translateX和translateY来进行设置，因为他们的百分比是自己的宽度或高度。

    <!DOCTYPE html>
<html>

<head>
    <style>
        html,
        body {
            height: 100%;
        }

        body {
            margin: 0;
        }

        body>div {
            width: 50%;
            height: 50%;
            border: 1px solid green;
            position: relative;
        }

        body>div>div {
            width:50%;
            height:50%;
            border:1px solid yellow;

            position:absolute;
            top:50%;
            left:50%;

            transform:translateX(-50%) translateY(-50%);
        }
    </style>
</head>

<body>
    <div>
        <div></div>
    </div>
</body>

</html>

小结

  1. translate中的百分比单位是相对于自身元素的 translate:(50%,50%);
  2. translate类似定位，不会影响到其他元素的位置
  3. 对行内标签没有效果

3. 2d旋转 rotate

2d旋转指的是让元素在2维平面内顺时针旋转或者逆时针旋转。

使用步骤：

  1. 给元素添加转换属性 transform
  2. 属性值为 rotate(角度) 如 transform:rotate(30deg) 顺时针方向旋转30度
div{
      transform: rotate(0deg);
}

在浏览器中手动修改 rotate

观察过后，2d旋转有以下特点

  1. 角度为正时顺时针，角度为负时为逆时针
  2. 默认旋转的中心点是元素的中心点

注意：

旋转时坐标也跟着在旋转，比如：

transform: rotatez(45deg) translateX(100px);

是按照旋转后x轴的方向移动位置的。

4. 2d缩放 scale

缩放，顾名思义，可以放大和缩小。 只要给元素添加上了这个属性就能控制它放大还是缩小

步骤：

  1. 给元素添加转换属性 transform
  2. 转换的属性值为 scale(宽的倍数,高的倍数) 如 宽变为两倍，高变为3倍 transform:scale(2,3)
div{
    transform:scale(2,3);
}

  • scaleX()，在x轴进行缩放。

  • scaleY()，在y轴进行缩放。

    scaleX和scaleY的值可以是实数（包括小数）或百分比，。值的意思是宽度或高度的倍数。

    注意：

    • 经过变形的元素可能比变形前大或小，不过元素在页面上所占据的空间与变形前保持不变。

      <!DOCTYPE html>
<html>

<head>
    <style>
        html,
        body {
            height: 100%;
        }

        body {
            margin: 0;
        }

        body>div {
            width: 50%;
            height: 50%;
            border: 1px solid green;
            position: relative;
        }

        body:hover div{
            transform:scaleX(1.2) scaleY(1.2);
        }
        /* body:hover>div{
            width:60%;
            height:60%;
        } */
    </style>
</head>

<body>
    <div>

    </div>
    <P>我为了证明所占的空间没变！</P>
</body>

</html>

  • scale(w,h)，在x轴和y轴上同时进行缩放。

    如果使用scale使用一个值时这个值将会同时应用在x轴和y轴上。

  2. transform:scale(2,2) 宽和高都放大了2倍
  3. transform:scale(2) 只写一个参数 第二个参数则和第一个参数一样 相当于 scale(2,2)
  4. transform:scale(0.5,0.5) 缩小为原来的一半

5. 转换中心 transform-origin

该属性可以修改元素旋转/缩放时候的中心点

  1. transform-origin:50% 50%; 默认值 元素的中心位置 百分比是相对于自身的宽度和高度
  2. transform-origin:top left; 左上角 和 transform-origin：0 0;相同
  3. transform-origin:50px 50px; 距离左上角 50px 50px 的位置
  4. transform-origin：0; 只写一个值的时候 第二个值默认为 50%;

6. 扭曲skew

格式：transform:skew(10deg, 10deg);

skew单位是deg，只写一个值，比如skew(10deg)时代表skew(10deg, 0)。

其他方法

  • skewX()，元素在水平方向扭曲。如果只为正数，元素的左上角和右下角将会被拉扯。如果值为负数，元素的右上角和左下角会被拉扯

    倾斜一个元素的同时也会倾斜元素的坐标系。

  • skewY()，元素在垂直方向扭曲变形。如果只为正数，元素的左上、右下将会被拉扯。如果只为负数，元素的右上、左下将会被拉扯。

二、 动画 animation

初学者容易对 动画 和 过渡 傻傻分不清楚。

过渡是在状态发生变化时把属性的值从一个规则中设定的值变为另外一个规则中设定的值，这个变化是在一段时间内完成的，不是瞬间变化的。

CSS动画与过渡的相同点就是CSS属性的值在一段时间内发生变化，不同点是动画对变化的方式有更大的控制权（如何重复，重复多少次），这个过渡从某种程度上来说只不过是效果的增强，过渡可以看成是一种简单的动画。

过渡只能看到一次变化过程 如宽度 1000px 变化到 100px ，而动画可以设置变化的次数甚至是无数次。

过渡通常需要配合hover使用，而动画不需要，可以自动执行。

1. 简单动画基本步骤

  1. 在css中定义动画函数

    要想给元素添加动画效果要有一个动画名，这个名字可以重复使用。

    @keyframes 动画名{

}

    关键帧的作用用来设置“由这个状态变成那个状态”。

    在规则中要有一个或多个关键帧块，每个关键帧的块包括关键帧选择符（用来表示动画时间内的时间点）和属性、值。多个关键帧块组成了一个完整的动画效果。

    @keyframes myTest{
  0%{
    property1:value1;
  }
  100%{
    property2:value2;
  }
}
    • 0%、100%表示动画播放到0%和100%的时候分别希望某个元素的属性值是什么。
    • 75%，表示动画持续时间执行到75%的时候的值。

  2. 给目标元素调用动画函数

    格式： animation-name:value

    • none，没有规则名称（默认），没有动画效果。
    • 规则名，使用@keyframes定义的动画名。

3.指定动画的持续时间。

只将动画应用到元素上不足以让动画呈现出来，动画虽然能够播放但是瞬间结束。所以要让动画持续一段时间。

格式：animation-duration:value

指定的持续时间必须带上单位，s或ms。

    /* 1 声明动画函数 */

    @keyframes ani_div {
      0%{
        width: 100px;
        background-color: red;
      }
      50%{
        width: 150px;
        background-color: green;
      }
      100%{
        width: 300px;
        height: 300px;
        background-color: yellow;
      }
    }

    div {
      width: 200px;
      height: 200px;
      background-color: aqua;
      margin: 100px auto;
      /* 2 调用动画 */
      animation-name: ani_div;
      /* 持续时间 */
      animation-duration: 2s;
    }

2. 动画进阶

  1. 速度曲线

    和设置过渡的速度曲线一样 animation-timing-function:linear;

    • linear： 匀速
    • ease： 慢-快-慢 默认值
    • ease-in： 慢-快。
    • ease-out： 快-慢。
    • ease-in-out： 慢-快-慢。
    • steps(n)：将动画时长分为n段，一段一段地展示执行动画，没有过渡效果。

  2. 延迟时间

    animation-delay: 0s;

  3. 循环次数

    设置动画播放的循环次数 animation-iteration-count: 2; infinite 为无限循环

  4. 循环方向

    animation-direction

    如在动画中定义了 0%：红色 100%：黑色 那么 当属性值为

    1. normal 默认值 红 -> 黑
    2. reverse 反向运行 黑 -> 红
    3. alternate 正-反-正... 红 -> 黑 -> 红...
    4. alternate-reverse 反-正-反.. 黑 -> 红 -> 黑 ...
    5. 第3点、第4点循环次数必须两次以上才有效果

  5. 动画等待或者结束的状态

    animation-fill-mode 设置动画在等待或者结束的时候的状态

    • forwards：动画结束后，元素样式停留在 100% 的样式
    • backwards：在延迟等待的时间内，元素样式停留在 0% 的样式
    • both：同时具备forwards和backwards两个属性值的特点

    区别：backwards表示等待期为第一帧样式，完成期跳转为初始样式；

    ​ forwards表示等待期保持初始样式，完成期保持最后一帧样式；

    ​ both表示等待期样式为第一帧样式，完成期保持最后一帧样式；

  6. 暂停和播放

    animation-play-state 控制 播放 还是 暂停

    running 播放 paused 暂停

3. 复合写法

animation: name duration timing-function delay iteration-count direction fill-mode;

案例：逐个显示文字的效果

<style>
        div {
            overflow: hidden;
            font-size: 20px;
            width: 0;
            height: 30px;
            background-color: yellow;
            /* 让我们的文字强制一行内显示 */
            white-space: nowrap;
            /* steps 就是分几步来完成我们的动画 有了steps 就不要在写 ease 或者linear 了 */
            animation: w 4s steps(10) forwards;
        }

        @keyframes w {
            0% {
                width: 0;
            }
            100% {
                width: 200px;
            }
        }
</style>

<body>
    <div>我在尚硅谷等你！！！</div>
</body>
