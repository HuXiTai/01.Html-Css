一、渐变

概念

渐变指的是一个颜色到另外一个颜色的平滑过渡，它是图像。可以用在任何一个使用图像的地方。而且渐变没有固定尺寸。（一般来说我们会将渐变用在背景上）

分类

按应用场景，渐变分为线性渐变和径向渐变。

按循环次数，渐变分为循环渐变和非循环渐变。

1.1线性渐变

线性渐变（linear gradients），线性渐变指沿着一条线进行填充得到的渐变，贯穿整个元素的中心点的这条线称为梯度线。

格式：

background-image:linear-gradient([方向/角度],color1, color2...);  /*可以不写方向/角度，但不能不写颜色*/

  1. 简单的颜色值。你可以使用CSS中的任何的颜色值，包括rgba和transparent，颜色可以有两个或多个。如果有多个颜色则平均分配在梯度线上。

    background-image: linear-gradient(yellow,red,green);

  2. 角度的设置

    假设背景图像是矩形，那么在外面虚拟一个圆形出来，圆形就有角度（顺时针为0 ~ 180deg，逆时针为0 ~ -180deg，默认180deg），假设现在是45deg的时候，将会在圆形45deg的地方做一条线，这条线叫梯度线。

  1. 梯度线的颜色分配

    梯度线在分配的时候会在梯度线90度的地方分散出来多条虚拟的线条这些线条无限延伸。

  2. 方向

    • to bottom，默认，梯度线方向向下。
    • to top，梯度线方向向上。
    • to left，向左
    • to right，向右
    • to left top，左上角（不等于315deg）
    • to left bottom，左下角
    • to right top，右上角
    • to right bottom，右下角

  • 经过图像的中心点，把声明的角的相邻的两侧的角链接起来。

  • 梯度线与那条相邻角的对角线垂直，指向声明的角

  • 颜色值默认值

    默认的时候如果有多个颜色平均分配在梯度线上，如果设置了三种颜色，第一个颜色为0%，最后一个颜色在100%的位置，那么：

    background-image: linear-gradient(to right, yellow, red, green, blue);如果设置了四种颜色，则：

  • 指定颜色值的位置。

    可以指定px或%作为单位，在指定的位置插入一个颜色。让这些颜色不会平均分配在梯度线上。

    <!DOCTYPE html>
<htmL>

    <head>
        <style>
            div {
                width: 500px;
                height: 100px;
                border: 1px solid green;
                background-image: linear-gradient(to right, yellow 0%, red 10%, green 30%, blue 50%);
            }
        </style>
    </head>

    <body>
        <div>
        </div>
    </body>

</htmL>

    注意： 由这个例子可以看出来，如果排布的颜色不到梯度线的末端就会将最后一个颜色一直延伸下去。

1.2 重复线性渐变

重复性线性渐变repeating-linear-gradient，重复性线性渐变在梯度线上不断循环排布的颜色，最后一个颜色的长度用于定义图像的整体长度，超过该长度之后开始循环填充。

和linear-gradient的相同点：

他们的百分比都是以盒子的宽/高为标准；而加了background-size属性，就以该属性设定的宽高值为标准。

不同点：

repeating-linear-gradient(to right, yellow 0%, red 10%, green 30%, blue 50%);会将50%处作为下一个循环填充的开始；而linear-gradient(to right, yellow 0%, red 10%, green 30%, blue 50%);会将50%-100%全部填充为蓝色。

案例：

background-image: repeating-linear-gradient(-70deg,yellow 0%,yellow 2%,black 2%,black 8%,yellow 8%,yellow 10%);

1.3 径向渐变

径向渐变是沿着一个中心点像四周扩展颜色进行平滑的过度。

格式：

background-image:radial-gradient([大小][at point] color1 color2.....);

  1. 简单的颜色值。

    可以使用CSS中的任何类型的颜色值，包括rgba和transparent，颜色可以有两个或多个，如果有多个默认平均分配在梯度线上。

    默认时，径向渐变在中间显示，在正方形元素中径向渐变为圆形，在长方形元素中径向渐变为椭圆形。

  2. 梯度线，梯度线从渐变的中心向右延伸，渐变的其他的部分根据这根线来构建。

  1. 默认的时候，如果有多个颜色则平均分配在梯度线上。

    可以给颜色指定一个位置，改变分配方式，设定的这个就是梯度线上的位置。

    background-image: radial-gradient(yellow,red 20px ,green);/*表示在离圆心20px的地方加入红色。*/

background-image: radial-gradient(red 30%, green 70%, blue 100%)；/*表示在以盒子对角线为直径的圆内，离圆心距离30%的地方为红色，离圆心70%的地方渐变为绿色，离圆心100%的地方渐变为蓝色*/

  2. 大小，在径向渐变中，径向尺寸可以设置渐变图像的大小。

    有以下三种情况：1、设置一个值；2、设置两个值；3、设置两个值同时指定圆心的位置。

    一个值：得到的是圆形渐变，值为半径。

    background-image:radial-gradient(20px,yellow,green);

    由上面的例子可以看出，最后一个颜色将从梯度线的终点一直向外延伸。

两个值：得到的是椭圆形渐变，第一个值为椭圆的宽度的一半，第二个值为椭圆的高度的一半。

   background-image: radial-gradient(150px 100px, yellow, green);

两个值加指定圆心：at point默认时径向渐变的中心在元素的中心点，可以使用at point来进行更改。at point后面的值就是表示发生径向渐变的圆的圆心在大盒子里的位置。

   background-image:radial-gradient(50px at 30% 30%,yellow ,green);

​ 我们也可以同时使用多个渐变，多个渐变之间用逗号连接，如果多个渐变之间有层叠的情况发生，那么先写的在上面。

二、过渡

过渡（transition)是CSS3中具有颠覆性的特征之一，我们可以在不使用 Flash 动画或 JavaScript 的情况下，当元素从一种样式变换为另一种样式时为元素添加效果。

过渡动画： 是从一个状态 渐渐的过渡到另外一个状态

可以让我们页面更好看，更动感十足，虽然 低版本浏览器不支持（ie9以下版本） 但是不会影响页面布局。我们现在经常和 :hover 一起搭配使用。

使用步骤：

  1. 指定要添加效果的CSS属性。
  2. 指定效果的持续时间。

定义过渡的属性

格式：

 transition-property:value;

只有在该属性中定义的属性（比如宽、高、颜色等）才会以动画的形式持续一段时间而不是瞬间变化。

  • none，不过渡任何属性。

  • all，过渡所有的属性。

    一般来说属性能够使用数字值的一般都支持过渡，不能转换为数值的关键字不支持过渡。

  • 其他值的列表，以逗号分隔的属性的列表。

设置过渡的持续时间

格式：

transition-duration:value;

指定一个状态过渡到另外一个状态要经过多久。

  • 默认值是0，即没有过渡时间。
  • 值可以是s或ms，只能是正数，一般来说较好的体验值是100~200ms。
  • 值可以逗号分隔的时间长度列表。
    • 如果想让列表中的所有属性都持续一个时间那就写一个值。
    • 如果想让每个属性都持续不同的时间那就写一个时间的列表。

调整过渡的方式

正常情况下，进行过渡的样式是慢速开始，然后加速然后再慢下来。如果要调整，需要用以下属性。

格式：

transition-timing-function:value;

值：

  • ease：默认值，慢速开始，然后加速，再减速。

  • linear：整个过程匀速。

  • ease-in：逐渐加速。

  • ease-out：逐渐减速。

  • cubic-bezier()，指定一个三次方的贝塞尔曲线。地址为：https://cubic-bezier.com

延迟过渡

transition-delay属性指的是开始过渡的延迟。

格式：

transition-delay:value;

值：和transition-duration的规则相同。

简写格式

语法：

添加一个动画，格式: transition:property duration timing-function delay
                 transition: 要过渡的属性  花费时间  运动曲线  何时开始;

添加多个动画，格式: transition:property1 duration1 timing-funtion1 delay1,
                property2 duration2 timing-function2 delay2.....
  • 属性 ： 想要变化的 css 属性， 宽度高度 背景颜色 内外边距都可以 。如果想要所有的属性都变化过渡， 写一个all 就可以
  • 花费时间： 单位是 秒（必须写单位） 比如 0.5s
  • 运动曲线： 默认是 ease （可以省略）
  • 何时开始：单位是 秒（必须写单位）可以设置延迟触发时间 默认是 0s （可以省略）
  • 后面两个属性可以省略
  • 记住过渡的使用口诀： 谁做过渡给谁加

反向过渡

​ 刚才只在div:hover上面声明了过渡，那么只在鼠标移入时才会触发过渡，移出时瞬间变为原来的状态。要解决这个问题，我们需要将过渡应用在默认状态上，然后由hover触发。

<!DOCTYPE html>
<html>

<head>
    <style>
        div {
            width: 100px;
            height: 20px;
            border: 1px solid green;
            background-color: green;
            transition: width 3s linear 100ms;
        }

        div:hover {
            width: 500px;
            /*  height:100px; */

            /* transition-property:width,height;
            transition-duration: 1s,2s; */

            /* transition-property: width;
            transition-duration: 3s; */
            /* transition-timing-function: cubic-bezier(.97,-0.63,0,1.49); */
            /* transition-timing-function: linear;
            transition-delay: 1s; */
        }
    </style>
</head>

<body>
    <div id="f">
    </div>
</body>

</html>
案例1：

步骤：

  • 创建两个div的盒子，属于的嵌套关系，外层类名叫 bar，里层类名叫 bar_in
  • 给外层的bar 这个盒子设置边框，宽高，圆角边框
  • 给里层的bar_in 设置 初试的宽度，背景颜色，过渡效果
  • 给外层的 bar 添加 hover事件，当触发了hover事件 让里层的bar_in 来进行宽度的变化

代码：

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title></title>
        <style type="text/css">
            .bar {
                width: 200px;
                height: 10px;
                border: 1px solid red;
                border-radius: 6px;
                margin: 20px auto;
            }

            .bar:hover .bar_in {
                width: 150px;
            }

            .bar_in {
                width: 0;
                height: 10px;
                background-color: red;
                /* 谁做过渡给谁加 */
                transition: width 3s;
            }
        </style>
    </head>
    <body>
        <div class="bar">
            <div class="bar_in"></div>
        </div>
    </body>
</html>
